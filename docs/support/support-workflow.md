# Support Workflow

## How to report a bug

Use the GitHub issue templates (see `.github/ISSUE_TEMPLATE/`) and include:
- what you expected vs what happened
- steps to reproduce
- device + OS + browser
- screenshots (avoid sensitive content)

## How to request help

If you cannot share details publicly, follow the security policy in `SECURITY.md` for sensitive issues.

## Data deletion requests

At the time of writing, there is no in-app account deletion UI. Data deletion requests should be handled by a maintainer with database access.
