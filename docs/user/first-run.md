# First-Run Guide

## 1) Create an account

On the landing screen, select **Sign Up** and create an account.

## 2) Complete onboarding

Onboarding captures the defaults used when generating weekly plans:
- household size
- trip type and budget
- preferred stores
- meal coverage and cooking style
- dietary rules and exclusions

You can change these later in **Settings**.

## 3) Add inventory and required items

Before generating a plan, add:
- **Inventory**: items you already have (pantry/fridge/freezer)
- **Lists / Required items**: staples you always need (or are currently out of)

## 4) Generate “This Week”

Go to **This Week** and generate a plan. The results are editable:
- remove recipes
- replace a recipe with a new suggestion
- review the grocery list / household list
