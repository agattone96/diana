# Uninstall Guide

## Web

- Close the tab or remove the site from your browser shortcuts.
- Clearing site data/cookies will remove local session storage.

## iOS / Android

- Uninstall the app as you would any other app.
- This removes locally stored data (including the session token).

## Account / backend data

Uninstalling the app does not automatically delete backend data. For deletion requests, see `docs/support/support-workflow.md`.
