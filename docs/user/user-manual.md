# User Manual

## Screens and what they do

### Landing / Sign Up / Log In

- Create an account or log in.
- Your session token is stored on-device for convenience.

### This Week

- Adjust weekly preferences (budget, trip type, meal coverage, cooking style).
- Generate a plan using inventory + required items + defaults.

### Inventory

- Add items manually (quantity, unit, location).
- Optionally extract items from a photo (if enabled in your deployment).

### Lists

- Manage “required items” (grocery/household staples).
- Review generated list output after creating a plan.

### Plan Results

- View recipe cards and a grouped list of items to buy.
- Remove a recipe or request a replacement recipe.

### Settings

- Update household defaults used by the planner.
- Reset defaults if needed.
- Log out.

## Data persistence

Your data is stored in the backend database per account:
- household defaults (profile)
- inventory items
- required items
- current weekly plan
- weekly history (when saved)
