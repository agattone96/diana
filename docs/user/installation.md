# Installation Guide

## What you need

- A supported browser (for the web build) or a mobile device (for iOS/Android builds).
- Internet access to the backend API.

## Web (recommended for quick evaluation)

Your deployment may be hosted on Netlify. Ask the project maintainer for the current URL.

If you are running locally as a developer, follow `docs/developer/local-development.md`.

## iOS / Android

This repository uses Expo. Distribution options vary by environment:
- Internal testing: Expo development build or TestFlight / Play Store internal track
- Production: app store builds from a signed release pipeline

If you received an install link from the team, follow that link and install the build.
